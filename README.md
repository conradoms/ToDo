   <h2 align="center">How to Use 🤔</h2>

   ```   
   - Clone this repository:
   $ git clone https://github.com/felipecastrosales/ToDo-Flutter

   - Enter in directory:
   $ cd ToDo-Flutter

   - For install dependencies:
   $ flutter packages get

   - Run the app: 
   $ flutter run
   ```
